|テーブル名|説明|
|---|---|
|users|登録されているユーザーの情報を格納する。ログイン処理の際にはこのテーブルからIDとパスワードを取得して処理を実行する。|
|project|登録されているプロジェクトを保存するテーブル。|
|task|登録されたタスクを保存するテーブル。|
|story|登録されたストーリーを保存するテーブル。|
